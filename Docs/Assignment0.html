Some terminology

Shell: A program that processes user commands and communicates with the operating system. For example, Bash (commonly used in Linux) or PowerShell (used in Windows) run commands and execute scripts.

Terminal: A software application that provides a text-based interface to the shell. Examples include GNOME Terminal (Linux) or Windows Terminal, which host shells like Bash or PowerShell.

Command Line: The specific input area within a terminal where users type commands to be executed by the shell.
