Bash Shell Reference

Following is the complete Bash reference from GNU. This is a comprehensive documentation detailing Bash shell features, command syntax, scripting capabilities, and built-in functions.

Following quick reference providing a categorized overview of the most commonly used Bash commands.

Commands are grouped into seven functional categories: File and Directory Management, Process Management, File Permissions and Ownership, Networking and Communication, Disk and System Info, Text Processing, and User Management.

Each table lists commands with descriptions, key options, and usage examples.

File and Directory Management

Command Description Options Type
ls List directory contents -l (long format), -a (show hidden), -h (human-readable) External
cd Change the current directory N/A Built-in
mkdir Create a new directory -p (create parents), -m (set permissions) External
rm Remove files or directories -r (recursive), -f (force deletion) External
cp Copy files or directories -r (recursive), -u (update only) External
mv Move or rename files or directories -i (prompt before overwrite), -u (update only) External

Process Management

Command Description Options Type
ps Display active processes -e (all processes), -f (full format) External
top Show real-time process information -d (delay interval), -u (user filter) External
kill Terminate a process by PID -9 (force kill), -15 (graceful stop) Built-in
bg Resume a job in the background N/A Built-in
fg Bring a background job to the foreground N/A Built-in
jobs List active jobs N/A Built-in

File Permissions and Ownership

Command Description Options Type
chmod Change file permissions -R (recursive), -v (verbose) External
chown Change file ownership -R (recursive), --reference (use another file's settings) External
umask Set default file permissions N/A Built-in
stat Display detailed file information -f (filesystem), -t (terse output) External

Networking and Communication

Command Description Options Type
netstat Display network connections and statistics -t (TCP), -u (UDP) External
nslookup Query DNS records N/A External
ping Check network connectivity -c (count), -i (interval) External
curl Transfer data from URLs -O (output to file), -L (follow redirects) External
wget Download files from the web -q (quiet), -r (recursive) External
ssh Remote login to a server -l (username), -p (port) External
scp Securely copy files between systems -r (recursive), -P (port) External

Disk and System Info

Command Description Options Type
df Show disk space usage -h (human-readable), -T (file system type) External
du Show directory size -h (human-readable), -s (summary) External
uptime Show system uptime N/A External
uname Display system information -a (all info), -r (kernel release) External
free Show memory usage -h (human-readable), -t (totals) External

Text Processing

Command Description Options Type
cat Display file contents -n (number lines), -E (show ends) External
echo Print text to the terminal -e (enable escape), -n (no newline) Built-in
grep Search for text patterns -i (ignore case), -v (invert match) External
awk Text processing tool -F (field separator), -v (assign variable) External
sed Stream editor for text manipulation -e (script), -i (in-place) External

User Management

Command Description Options Type
whoami Show current user N/A Built-in
id Display user ID -u (user ID), -g (group ID) External
who Show who is logged in -b (last boot), -u (users) External
passwd Change user password -d (delete password), -l (lock account) External
su Switch user - (login shell), -c (run command) Built-in
