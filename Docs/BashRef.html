Bash Shell Commands Cheat Sheet

This quick reference provides a categorized overview of commonly used Bash commands.

Commands are grouped into seven functional categories: File and Directory Management, Process Management, File Permissions and Ownership, Networking and Communication, Disk and System Info, Text Processing, and User Management.

Each table lists commands with descriptions, key options, usage examples, and the underlying system calls invoked when the commands are executed.

This classification ensures that similar commands are grouped by their core functionalities.

The structure enables a practical and organized reference for Bash users.

Quick Reference to Bash System Commands

1. File and Directory Management

Bash Command Description Options Usage Example System Calls (Under the Hood)
ls List files in directory -l (long format), -a (show hidden files) ls -la /home open(), readdir()
cd Change directory None cd /home/user chdir()
mkdir Create directory -p (create parent directories) mkdir -p /home/user/new_folder mkdir()

2. Process Management

Bash Command Description Options Usage Example System Calls (Under the Hood)
ps View running processes -aux (all processes, details) ps aux read() from /proc/
kill Terminate process -9 (force kill) kill -9 1234 kill()

3. File Permissions and Ownership

Bash Command Description Options Usage Example System Calls (Under the Hood)
chmod Change file permissions -R (recursive) chmod -R 755 file chmod()
chown Change file ownership -R (recursive) chown -R user file chown()

4. Networking and Communication

Bash Command Description Options Usage Example System Calls (Under the Hood)
ping Test network connectivity -c (count), -i (interval) ping -c 4 google.com socket(), sendto()
curl Fetch URLs -o (output file), -I (headers only) curl -o file.txt https://example.com connect(), send()

5. Disk and System Info

Bash Command Description Options Usage Example System Calls (Under the Hood)
df Disk space usage -h (human-readable) df -h statfs()
du Disk usage of directories -s (summary), -h (human-readable) du -sh folder stat()

6. Text Processing

Bash Command Description Options Usage Example System Calls (Under the Hood)
cat Display file contents -n (number lines) cat -n file.txt read(), write()
grep Search text in files -i (ignore case), -r (recursive) grep -i "text" file.txt read()

7. User Management

Bash Command Description Options Usage Example System Calls (Under the Hood)
who Show logged-in users None who read() from /var/run/utmp
id Show user ID info None id username getuid(), getgid()
