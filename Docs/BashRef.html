Bash Shell Reference

Following is the complete Bash reference from GNU. This is a comprehensive documentation detailing Bash shell features, command syntax, scripting capabilities, and built-in functions.

Following quick reference providing a categorized overview of the most commonly used Bash commands.

Commands are grouped into seven functional categories: File and Directory Management, Process Management, File Permissions and Ownership, Networking and Communication, Disk and System Info, Text Processing, and User Management.

Each table lists commands with descriptions, key options, and usage examples.
