Bash Shell Reference

Introduction

Following is a quick reference providing a categorized overview of the most commonly used Bash commands.

Commands are grouped into seven functional categories: File and Directory Management, Process Management, File Permissions and Ownership, Networking and Communication, Disk and System Info, Text Processing, and User Management.

Bash is a Unix shell and command interpreter that processes user commands and interacts with the operating system. A shell is a command-line interface that interprets commands, runs applications, and manages system tasks by communicating with the operating system’s kernel.

Bash serves two main roles: a command interpreter and a scripting language. As a command interpreter, it processes and executes individual user commands, enabling tasks like file management, process control, and system monitoring. This reference focuses on its command interpreter functionality.

Bash follows POSIX (Portable Operating System Interface) compliance, ensuring that its commands and scripts are portable across different Unix-like operating systems. Relevant parts of the POSIX standard for shells include command interpretation, process control, input/output redirection, job control, and shell scripting syntax. While Bash includes advanced features beyond POSIX, adherence to this standard guarantees compatibility and consistent behavior across various environments.

In the following, each table lists some commands within a category, along with brief descriptions, available options, and usage examples.

The complete Bash reference from GNU is given at the end of this page.

File and Directory Management and Editing

Command Description Options Type Usage
ls List directory contents -l (long format), -a (show hidden), -h (human-readable) External ls -l /path/to/directory
pwd Prints the present working directory None Built-in pwd
touch Creates an empty file or updates the timestamp of an existing file -a (change access time only), -m (change modification time only), -t (specify a timestamp) External touch filename.txt
cd Change the current directory N/A Built-in cd /path/to/directory
mkdir Create a new directory -p (create parents), -m (set permissions) External mkdir -p /path/to/directory
rm Remove files or directories -r (recursive), -f (force deletion) External rm -rf /path/to/directory
cp Copy files or directories -r (recursive), -u (update only) External cp -r /source/path /destination/path
mv Move or rename files or directories -i (prompt before overwrite), -u (update only) External mv /source/path /destination/path
less/more Displays file contents one screen at a time -N (show line numbers), -q (quiet mode), -S (disable line wrapping) External less filename.txt / more filename.txt
head Displays the first few lines of a file -n (number of lines to display), -c (number of bytes to display) External head filename.txt
tail Displays the last few lines of a file -n (number of lines to display), -f (follow file updates in real time) External tail filename.txt
nano Opens a simple text editor -B (create a backup file), -m (enable mouse support), -c (show cursor position) External nano filename.txt

Process Management

Command Description Options Type Usage
ps Display active processes -e (all processes), -f (full format) External ps -ef
top Show real-time process information -d (delay interval), -u (user filter) External top -d 3
kill Terminate a process by PID -9 (force kill), -15 (graceful stop) Built-in kill -9 1234
bg Resume a job in the background N/A Built-in bg %1
fg Bring a background job to the foreground N/A Built-in fg %1
jobs List active jobs N/A Built-in jobs

File Permissions and Ownership

Command Description Options Type Usage
chmod Change file permissions -R (recursive), -v (verbose) External chmod -R 755 /path/to/directory
chown Change file ownership -R (recursive), --reference (use another file's settings) External chown -R user:group /path/to/directory
umask Set default file permissions N/A Built-in umask 022
stat Display detailed file information -f (filesystem), -t (terse output) External stat /path/to/file

Networking and Communication

Command Description Options Type Usage
ifconfig Displays or configures network interfaces (deprecated in some systems) -a (show all interfaces), up (activate an interface), down (deactivate an interface) External ifconfig
ip Displays and manages network interfaces and routing addr (show IP addresses), link (show network interfaces), route (show routing table) External ip addr
netstat Display network connections and statistics -t (TCP), -u (UDP) External netstat -tu
nslookup Query DNS records N/A External nslookup example.com
ping Check network connectivity -c (count), -i (interval) External ping -c 4 example.com
curl Transfer data from URLs -O (output to file), -L (follow redirects) External curl -O https://example.com/file.zip
wget Download files from the web -q (quiet), -r (recursive) External wget -q https://example.com/file.zip
ssh Remote login to a server -l (username), -p (port) External ssh -l user example.com
scp Securely copy files between systems -r (recursive), -P (port) External scp -r file.txt user@example.com:/remote/path

Disk Operations

df Displays available disk space -h (human-readable), -T (file system type) External df -h
du Shows disk usage of files and directories -h (human-readable), -s (summary) External du -h filename
mount Mounts a file system -t (file system type), -o (mount options) External mount /dev/sdX /mnt
umount Unmounts a file system -f (force unmount), -l (lazy unmount) External umount /mnt
mkfs Formats a partition -t (specify file system type) External mkfs -t ext4 /dev/sdX
blkid Displays block device information None External blkid
fdisk Partition management utility -l (list partitions) External fdisk -l
parted Manages disk partitions -l (list devices), mkpart (create partition) External parted /dev/sdX
lsblk Displays information about block devices -f (file system info), -o NAME,SIZE (custom output) External lsblk -f
tune2fs Adjusts ext2/3/4 file system parameters -c (max mount count), -i (interval between checks) External tune2fs -c 1 /dev/sdX
sync Flushes file system buffers None Built-in sync

Text Processing

Command Description Options Type Usage
cat Display file contents -n (number lines), -E (show ends) External cat -n file.txt
echo Print text to the terminal -e (enable escape), -n (no newline) Built-in echo -e "Hello\nWorld"
grep Search for text patterns -i (ignore case), -v (invert match) External grep -i "pattern" file.txt
awk Text processing tool -F (field separator), -v (assign variable) External awk -F ":" '{print $1}' file.txt
sed Stream editor for text manipulation -e (script), -i (in-place) External sed -i 's/old/new/g' file.txt
sort Sorts lines of text files -r (reverse the sort order), -n (numeric sort), -u (unique sort) External sort filename.txt
uniq Removes duplicate lines from sorted input -c (prefix lines with count), -d (display only duplicates), -u (display unique lines) External uniq filename.txt
wc Counts lines, words, and characters in a file -l (count lines), -w (count words), -c (count bytes/characters) External wc filename.txt

User Management and System Information

Command Description Options Type Usage
whoami Show current user N/A Built-in whoami
id Display user ID -u (user ID), -g (group ID) External id -u
who Show who is logged in -b (last boot), -u (users) External who -u
passwd Change user password -d (delete password), -l (lock account) External passwd user
su Switch user - (login shell), -c (run command) Built-in su - user
sudo Runs a command with superuser privileges -u (run as a specific user), -k (reset cached credentials), -s (run shell as root) External sudo command_name
uname Displays system information -a (all system info), -s (kernel name), -r (kernel release), -m (machine hardware name) External uname -a
uptime Displays system uptime and load averages -p (pretty format), -s (system boot time) External uptime
hostname Displays or sets the system's hostname -f (full hostname), -i (IP address), -d (domain name) External hostname
